# To-Do-List WebApp

This project was built using:
-ReactJS
-NodeJS
-Express
-MongoDB

# Dependencies

cors: 2.8.5
express: 4.17.3
mongoose: 6.2.2
nodemon : 2.0.15

# Client Commands
    -npm install
    -npm start

If not working try: 

1-delete package-lock.json and node_modules and then run "npm install"

2-try these :

      npm cache clean
      npm init
      npm install
      
 # Server Commands 
    
    npm install 
    npm run devStart
